# Routes

## **Realtor App routs**

| **Method** | **Path**  | **Action**           |
| ---------- | --------- | -------------------- |
| GET        | /Home     | Get All Home         |
| GET        | /Home/:id | GET One Home         |
| POST       | /Home     | Create a Home        |
| PUT        | /Home/:id | Update a Home        |
| DELETE     | /Home/:id | Delete a Home        |
| POST       | /Home/:id | Inquire about a home |
